Elastic Layout

An elastic layout uses ems for widths.

This enables containers to grow proportionally to text, when the text is zoomed.

Achieves WCAG 2.0 Criterion 1.4.4.(Resize text) - Technique page

width: 60em,
margin: 0.625em
