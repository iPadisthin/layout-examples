Fixed Layout

A fixed layout has a fixed width outer container.

Containers within it can use pixels, percentages or ems for width

width: 980px

width: 960px,
margin: 10px

width: 520px,
margin: 10px

width: 200px,
margin: 10px

width: 200px,
margin: 10px

Even if inner containers use percentages (or ems) for width, its still a fixed width layout because outer container fixed.
